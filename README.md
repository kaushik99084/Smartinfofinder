Smart Info Finder is an AI-powered chatbot web application built using Streamlit and Groq API.
The project provides fast, accurate, and multilingual responses using the Llama 3.3 70B Versatile model.

Users can ask questions in any language, and the chatbot automatically replies in the same language, creating a smooth and user-friendly experience.

🚀 Features
🌍 Multilingual Support
Supports English, Hindi, Punjabi, and many other languages.
Automatically detects and responds in the user's language.
🤖 AI-Powered Responses
Uses Groq’s ultra-fast inference with the llama-3.3-70b-versatile model.
💬 Interactive Chat Interface
Real-time conversation using Streamlit chat components.
🗑️ Clear Chat History
Users can reset the conversation anytime.
🎨 Custom UI Styling
Clean and modern interface with custom CSS styling.
⚡ Fast Performance
Optimized for quick AI responses.
